This plugin gives some statistics about your posts (word count, etc.)

== Description ==

This plugin will show stats about your posts after your content.

Currently it shows:

 * Word count

I plan to add:

 * Characters count
 * Reading time
 * Speaking time

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/postats` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Stats will show after your posts.
